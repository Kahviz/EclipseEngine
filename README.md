# BoronEngine
  A My GameEngine That I have been devoleping since 12.11.2025
  It has two renderers Vulkan- And DirectX11Renderer
  Supports only Windows. Linux support is coming soon!
  It's in DEV so it isn't perfect.
# Libraries That I Used
  BoronEngine Uses Assimp to Load Meshes, SDL3 to make controller effects like rumble, ImGui to Make Fantastic GUI and stb_image to load it's icons.
  
# How to
  Switch A renderer engine?
    -Go To GLOBALS.h and you see 3 #defines if you want to use DX11 put DIRECTX11 to 1 and if you don't want to use    
    DX11 put that into 0.The same thing is in the Vulkan.
    
# Building For VS Community 2026
1. Download the zip-file or clone the repo
2. Go File Onto Top Right and press Open And Then CMake Project
3. Press The Run Button and wait about half minute So It can clone all the resources from The Repo to AppData Folder
   
# Requirements
1. Vulkan SDK
2. Windows
3. Vs Commynity 2026
4. CMake
