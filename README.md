# Libraries That I Used
  UntilitedGameEngine Uses Assimpt to Load Meshes, SDL3 to make controller effects like rumble, ImGui to Make Fantastic    GUI and STBIcons to load it's icons.
  
# How to
  Switch A renderer engine?
    -Go To GLOBALS.h and you see 3 #defines if you want to use DX11 put DIRECTX11 to 1 and if you don't want to use         DX11 put that into 0.The same thing is in the Vulkan.
# Building For VS Commynity 2026
1. Down load the zip file or clone the repo
2. Go File Onto Top Right and press Open And Then CMake Project
3. Press The Run Button and wait about half minute So It can clone all the resources from The Repo to AppData Folder
   
+You need Vulkan SDK
